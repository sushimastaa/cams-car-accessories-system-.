┌─────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

❌ "Connection Refused"
   → Start Apache: New folder\apache_start.bat
   → Start MySQL: New folder\mysql_start.bat

❌ "Page Not Found"
   → Check files are in: New folder\htdocs\cams\
   → Run: SETUP_PROJECT.bat

❌ "Database Connection Error"
   → Create database 'cams_db' in phpMyAdmin
   → Ensure MySQL is running

❌ "Port 80 in Use"
   → Change to port 8080 in New folder\apache\conf\httpd.conf
   → Access via: http://localhost:8080/cams/
